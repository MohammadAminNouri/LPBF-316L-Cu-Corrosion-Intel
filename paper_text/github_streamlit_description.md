# GitHub / Streamlit Description for the Paper

## Short GitHub description

Reproducible Streamlit dashboard for ML-assisted electrochemical fingerprinting of LPBF AISI 316L and 316L-Cu in NaCl and H2O2/NaCl simulated inflammatory electrolytes.

## Longer project description

This repository accompanies a study on the electrochemical and surface degradation behavior of laser powder bed fusion AISI 316L and Cu-bearing AISI 316L stainless steel exposed to normal chloride and peroxide-containing simulated inflammatory electrolytes. The dashboard converts the manuscript's polarization and EIS parameter tables into a reproducible feature matrix, then applies transparent data-driven tools including corrosion severity indexing, PCA-based electrochemical fingerprinting, and nearest-prototype degradation-state classification.

The aim is not to claim a universal corrosion predictor. Instead, the project demonstrates how summarized electrochemical descriptors can be organized into an interpretable workflow that supports the mechanistic conclusion of the paper: peroxide shifts the electrochemical state toward a more oxidizing but less protective surface condition, while Cu-bearing LPBF 316L shows the highest degradation severity under H2O2/NaCl exposure.
